# 🛡️ CiberNova

**Protegemos tu mundo digital.**

Sitio web profesional de ciberseguridad construido con React + Vite.

## 🚀 Tecnologías

- React 19
- Vite
- CSS Vanilla (Design System personalizado)
- Google Fonts (Inter + Outfit)

## 📋 Secciones

- 🏠 Hero con partículas animadas
- 👥 Sobre Nosotros (Misión, Visión, Valores)
- 🛡️ Servicios de ciberseguridad
- 📊 Estadísticas con contadores animados
- ⚠️ Amenazas digitales comunes
- 💬 Testimonios (carousel)
- 📝 Blog de ciberseguridad
- 📧 Formulario de contacto
- 🔽 Footer

## ⚡ Instalación

```bash
npm install
npm run dev
```

## 📦 Build

```bash
npm run build
```
